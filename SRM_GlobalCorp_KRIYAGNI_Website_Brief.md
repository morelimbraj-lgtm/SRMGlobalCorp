# KRIYAGNI Website — SRM Global Corp

## Complete Brief for Designer / Developer

### Brand Direction
**Positioning:** Premium hospitality consulting and advisory brand for owners, investors, and operators.

**Overall Style:**
- Apple / Steve Jobs minimalism with hospitality warmth
- One-scroll homepage with very few supporting pages
- Large whitespace, calm pacing, confident typography
- One message per screen
- Premium, editorial, architectural feel

**Typography:**
- Headlines: SF Pro, Helvetica Neue, or similar clean grotesk
- Very large type for hero and section statements
- Body copy kept short and highly legible

**Color System:**
- Primary: white / off-white background
- Text: black / charcoal
- Accent: one subtle premium accent color only (deep gold, muted green, or warm terracotta)
- Avoid gradients, neon tones, and busy palettes

---

## Strategic Recommendation
This should not feel like a generic hotel website.

SRM Global Corp is a consulting and advisory company, so the website should communicate:
- authority
- clarity
- trust
- commercial outcomes
- process discipline
- premium taste

The best direction is a **hybrid of Apple minimalism + consulting credibility + hospitality visual richness**.

---

# Homepage Structure

## Section 1 — Hero
**Layout:**
- Full-screen height
- Centered headline
- Massive whitespace above and below
- Minimal top navigation

**Headline Options:**
- Hospitality strategy that moves revenue.
- Build sharper hospitality businesses.
- Consulting for hospitality that performs.

**Subtext:**
- Strategy, audits, franchise systems, and operating improvement for hospitality brands.

**CTA:**
- Explore services
- Book a consultation

**Design Notes:**
- Use a single strong image, short motion loop, or abstract hospitality visual
- Keep the first screen extremely clean
- No clutter, no extra badges, no long paragraph

---

## Section 2 — The Problem / Solution
**Layout:**
- Split-screen or editorial layout
- Left: operational chaos / scattered brand assets / poor systems
- Right: structured growth framework / clean dashboard / report-style visual

**Message:**
- Most hospitality businesses run on instinct.
- SRM Global Corp turns complexity into systems.

**Design Notes:**
- Use contrast between confusion and order
- This section should explain the value proposition instantly
- Visual should do most of the work

---

## Section 3 — What SRM Does
**Format:**
A refined grid of 4–6 service tiles with one-line descriptions.

**Suggested Services:**
- Hospitality audits
- Franchise strategy
- Business operating systems
- F&B growth strategy
- Turnaround consulting
- Brand and expansion planning

**Design Notes:**
- Each tile should feel premium and sparse
- Use hover states that reveal one extra line only
- Avoid long service paragraphs

---

## Section 4 — Proof / Credibility
**Format:**
- Full-width case study strip
- Metrics and outcomes
- Logos or client types if available
- Sample reports or audit previews

**Suggested Copy:**
- Real problems. Structured thinking. Measurable improvements.

**What to Show:**
- before/after snapshots
- a sample report
- process visuals
- selected project highlights

**Design Notes:**
- Use actual proof instead of generic claims
- Make the work feel tangible and useful

---

## Section 5 — The Method
**Headline:**
- Clear process. Better decisions.

**Format:**
A 3-step or 4-step vertical flow.

**Example:**
1. Diagnose
2. Strategize
3. Implement
4. Optimise

**Design Notes:**
- This should feel like a premium consulting playbook
- Use line art, numbered steps, or subtle animation
- Keep copy extremely lean

---

## Section 6 — Authority / Numbers
**Format:**
Large centered figures with fade-in animation on scroll.

**Example Numbers:**
- 14+ years experience
- 40+ completed projects
- 4+ countries
- 100+ assignments

**Design Notes:**
- Numbers should appear one by one
- Big type, small labels
- Use ample whitespace so each metric feels significant

---

## Section 7 — Final CTA
**Headline:**
- Let’s build a stronger hospitality business.

**Buttons:**
- Request access
- Start a conversation

**Design Notes:**
- Keep the CTA simple and calm
- No aggressive sales language
- Make it easy to contact the team
