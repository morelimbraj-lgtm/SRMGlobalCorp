# SRM Global Corp Site

Static site scaffold for the SRM Global Corp hospitality advisory brief.

## Stack

- Multi-page HTML
- Shared CSS and small vanilla JavaScript interactions
- Vite for local development and production builds

## Pages

- `/` homepage
- `/services.html`
- `/research.html`
- `/contact.html`

## Run

```bash
npm install
npm run dev
```

## Notes

- Copy, metrics, and service structure are based on `SRM_GlobalCorp_KRIYAGNI_Website_Brief.md`.
- Contact form action is currently a placeholder and should be wired to email, CRM, or a form backend.
